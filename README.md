# PIDscope

PIDscope is a set of graphical tools for analyzing blackbox log data for multirotors. It runs on GNU Octave (Windows, Mac, Linux) and is tailored to work on Betaflight, Emuflight, INAV, FETTEC and QuickSilver logfiles. Originally based on PIDtoolbox by Brian White.
